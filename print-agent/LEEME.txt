# ParqueoSys — impresión para el PC del operador

## Recomendado (más cómodo)

### Una sola vez por PC
1. En la web, descarga en la misma carpeta:
   - `Instalar-ParqueoSys-Impresora.bat`
   - `Instalar-ParqueoSys-Impresora.ps1`
2. Ejecuta el `.bat`
3. El instalador descarga el agente, lo registra y lo pone al inicio de Windows

### Después
- Al encender el PC, la impresión suele estar lista
- Si no: en la web pulsa **Activar impresora** (abre el agente solo)
- Usa https://parqueo-two.vercel.app en ese mismo PC

## Requisitos
- Windows + impresora **LABEL** por USB + etiquetas 40×30 mm
